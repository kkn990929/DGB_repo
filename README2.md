## 필요한 라이브러리

  1) numpy
  2) pandas
  3) plotly
  4) missingno

## 설치방법

```
pip install numpy
```

만약 회사 내부 보안으로 설치가 안되면 다음 코드로 설치

```
pip --trusted-host pypi.org --trusted-host files.python hosted.org install numpy
```
